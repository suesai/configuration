## crx

1. 从 [crx4chrome](https://www.crx4chrome.com) 获取 id
2. 从 [crxdown](https://crxdown.com/) 下载
3. 修改后缀为 rar，并解压
4. 扩展程序 -> 开发者模式 -> 加载已解压的扩展程序
